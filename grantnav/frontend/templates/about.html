{% trans "About GrantNav" %}

{% blocktrans %}GrantNav is 360Giving’s search-engine for grants data. It is a free-to-use platform that brings together open, comparable grants data published in the 360Giving Data Standard. GrantNav makes it easy to search, explore and download data across billions of pounds of grants given by a wide range of UK grantmakers.{% endblocktrans %}

{% blocktrans %}For the list of organisations featured in this platform, visit: https://grantnav.threesixtygiving.org/funders.{% endblocktrans %}

{% blocktrans %}New data is added to GrantNav as more organisations publish more of their grants in the 360Giving Data Standard.{% endblocktrans %}

{% blocktrans %}GrantNav has been developed to help find out which organisations have funded which sectors or regions, so resources – and how they change over time – can be mapped. This helps funders to get a quick overview of other organisations working in the same areas.{% endblocktrans %}

{% blocktrans %}For help with searching and downloading (including advanced searches), visit: https://grantnav.threesixtygiving.org/help.{% endblocktrans %}

{% blocktrans %}For more information about GrantNav and the the data used in it, please see the links in the footer below.{% endblocktrans %}

{% blocktrans %}GrantNav was developed for 360Giving by Open Data Services. The project received funding from the Esmée Fairbairn Foundation, Indigo Trust and Nesta.{% endblocktrans %}

{% blocktrans %}We welcome feedback and questions on GrantNav. Please email: grantnav@threesixtygiving.org{% endblocktrans %}
